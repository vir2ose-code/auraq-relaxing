# ✧ AURAQ ✧ 
### Relaxing Music Production & Meditation App

![AURAQ Header](https://auraq-relaxing.com/auraq-icon-512.png)

**AURAQ** ist eine hochmoderne Progressive Web App (PWA), die speziell für tiefe Entspannung, Meditation und Heilschlaf entwickelt wurde. Die App kombiniert sphärische 432 Hz Kompositionen mit einem minimalistischen, edlen Design.

---

## ✨ Features

- **✧ 432 Hz Audio Experience:** Alle Tracks sind in der heilenden 432 Hz Frequenz produziert.
- **✧ Native PWA Feeling:** Als App auf Android und iOS installierbar mit Offline-Unterstützung.
- **✧ Multilingual Support:** Vollständig übersetzt in **Deutsch, Englisch, Polnisch und Spanisch**.
- **✧ Interactive UI:** Sanfte Animationen, Parallax-Sterne und haptisches Feedback für maximale Immersion.
- **✧ Meditation & Sound Healing:** Spezialisierte Sektionen für geführte Meditationen, Kristall-Klangschalen und Schlafabläufe.

---

## 🛠️ Technologie & Stack

- **Core:** HTML5, Vanilla CSS3, Modern JavaScript (ES6+)
- **Build Tool:** [Vite](https://vitejs.dev/)
- **Hosting:** [Vercel](https://vercel.com/)
- **Integration:** Web3Forms für das Kontaktformular.

---

## 🚀 Installation & Deployment

1. **Repository klonen:**
   ```bash
   git clone https://github.com/vir2ose-code/auraq-relaxing.git
   ```
2. **Abhängigkeiten installieren:**
   ```bash
   npm install
   ```
3. **Lokal starten:**
   ```bash
   npm run dev
   ```

---

## 📱 Mobile Installation (PWA)

Die App kann direkt über den Browser installiert werden:
1. Öffne [auraq-relaxing.com](https://auraq-relaxing.com) auf deinem Smartphone.
2. Tippe im Menü auf **"✦ APP INSTALLIEREN"** (Android) oder nutze "Zum Home-Bildschirm hinzufügen" (iOS).
